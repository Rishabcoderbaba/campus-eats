# Campus Eats

Campus Eats is a web application that allows students to browse, order, and manage food items from their campus cafeteria. It includes features for both users and admins, making food ordering simple, fast, and convenient.

## Features

- Browse food menu with categories
- Add items to cart and place orders
- User authentication (Signup/Login)
- Admin dashboard to manage menu and orders
- Responsive design for mobile and desktop

## Installation

1. Clone the repository:

```bash
git clone https://github.com/username/campus-eats.git

cd campus-eats

# For frontend
cd client
npm install

# For backend
cd ../server
npm install

# Backend
npm run dev

# Frontend (in another terminal)
cd ../client
npm start
